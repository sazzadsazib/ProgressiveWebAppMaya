  • face What is Maya Apa Plus?
    Ans:

    By Subscribing to Maya Apa Plus Users will enjoy the following Features -

    • check_circle 24/7 dedicated support from the Maya Experts on Health and psychosocial issues almost instantly.

    • check_circle Experts will respond to each and every question typically within 10 minutes.

    • check_circle Users will receive timely update on their last question status

    • check_circle Users will also have the feature of photo attachment and asking voice question. and

    • check_circle Users will be able to customize the look of the Maya Apa app.

  • file_download How do I install Maya Apa app?
    Ans:

    User can type any of these keywords based on his/her requirement of the type of subscription: START MayaApa 1/ START MayaApa 7/ START MayaApa 14/ START MayaApa 30 or START MayaApa 7AR/ START MayaApa 14AR/ START MayaApa 30AR and send it to 8008. In return SMS user will get the link of Maya Apa app download.
    or
    User can visit google Playstore OR search “Maya Apa” on Google Play Store. Once you find the app, click on the install button to install the Maya Apa app.
  • verified_user How to sign up?
    Ans:

    Open Maya Apa app and click on “Sign In” from the navigation drawer menu on the left. Then click on “SIGN UP IN LESS THAN A MINUTE”. After that complete sign up selecting one option from the three available options (sign up with email/sign up with mobile number/continue with facebook) and sign up. These steps are only for general of Maya Apa App users NOT for those users who upgrade or subscribe to Maya Apa Plus (Robi) users. Users will automatically be signed up if they subscribe by using Robi data.

  • textsms How to activate Maya Apa Plus Service through SMS?
    Ans:

    User can activate the Maya Apa Plus service through:

    With Auto-renewal

    SMS- STARTMayaApa30AR to 8008 (30 Days)
    SMS- STARTMayaApa15AR to 8008 (15 Days)
    SMS- STARTMayaApa7AR to 8008 (7 Days)

    With Auto-renewal

    SMS- STARTMayaApa30 to 8008 (30 Days)
    SMS- STARTMayaApa15 to 8008 (15 Days)
    SMS- STARTMayaApa7 to 8008 (7 Days)
    SMS- STARTMayaApa1 to 8008 (1 Days)
